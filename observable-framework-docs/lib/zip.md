# [ZIP](https://observablehq.com/framework/lib/zip\#zip)

To load a [ZIP archive](https://en.wikipedia.org/wiki/ZIP_(file_format)), use [`FileAttachment`](https://observablehq.com/framework/files).

```js
const muybridge = FileAttachment("muybridge.zip").zip();
```

This returns a promise to a `ZipArchive`. This uses [JSZip](https://stuk.github.io/jszip/) under the hood.

ZipArchive {filenames: Array(3)}

```js
muybridge
```

The `filenames` property lists the paths of the files contained within the ZIP archive.

Array(3) \["deer.jpeg", "horse.jpeg", "ox.jpeg"\]

```js
muybridge.filenames
```

To pull out a single file from the archive, use the `archive.file` method. It returns a `FileAttachment` which you can use to load the file contents like any other file.

![A deer](blob:https://observablehq.com/c43a9803-645c-4795-bc75-f4bc7b388d5b)

```js
muybridge.file("deer.jpeg").image({width: 320, alt: "A deer"})
```

That said, if you know the name of the file within the ZIP archive statically, you don’t need to load the ZIP archive; you can simply request the [file within the archive](https://observablehq.com/framework/data-loaders#archives) directly. The specified file is then extracted from the ZIP archive at build time.

![A deer](https://observablehq.com/framework/_file/lib/muybridge/deer.c41748b9.jpeg)

```js
FileAttachment("muybridge/deer.jpeg").image({width: 320, alt: "A deer"})
```

For images and other media, you can simply use static HTML.

![A deer](https://observablehq.com/framework/_file/lib/muybridge/deer.c41748b9.jpeg)

```html
<img src="muybridge/deer.jpeg" width="320" alt="A deer">
```

One reason to load a ZIP archive is that you don’t know the files statically — maybe there are lots of files and you don’t want to enumerate them statically, or maybe you expect them to change over time and the ZIP archive is generated by a [data loader](https://observablehq.com/framework/data-loaders). For example, maybe you want to display an arbitrary collection of images.

![](blob:https://observablehq.com/dd0b3791-56d8-4ea9-81f1-1dc105d1d864)![](blob:https://observablehq.com/a9d0268b-b034-4b7c-9b87-997ea7451058)![](blob:https://observablehq.com/99bee8fb-5122-4804-bd47-e87beab470b2)

```js
Gallery(await Promise.all(muybridge.filenames.map((f) => muybridge.file(f).image())))
```

```js
function Gallery(images) {
  return html`<div style="
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 1rem;
">${images.map((i) => (i.style = 'width: 100%; height: 100%;', i))}</div>`;
}
```

To let the user download a ZIP archive, use the `download` attribute on a link.

[download zip](https://observablehq.com/framework/_file/lib/muybridge.8a53b0d6.zip)

```html
<a href="muybridge.zip" download>
  <button>download zip</button>
</a>
```
